# Test Task

## installation
```bash
    $ composer install
    
    $ npm install
```
## Setup project

```bash
    $ cp .env.example .env
    
    $ php artisan key:generate
    
    $ php artisan storage:link
    
    $ php artisan migrate --seed
```

## Run application

```bash
    $ php artisan serve
    
    $ npm run dev
```
